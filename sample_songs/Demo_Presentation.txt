Slide 1: Introduction
This plugin supports text files
Where each line is a slide

Slide 2: Features
- Background images
- Custom text colors
- Borders and shadows
- Easy navigation

Slide 3: Usage
Simply place txt files in a folder
Select the folder in OBS
Choose your song/presentation
Navigate with the controls

Slide 4: Customization
Adjust font size and style
Pick your favorite colors
Add borders for emphasis
Create shadows for depth

Slide 5: Thank You
Enjoy using the plugin!
